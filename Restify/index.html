Creating a Server


					var restify = require('restify');

					var server = restify.createServer({
    				name: 'MyApp'
					});

					server.listen(8080);

server.use()

Need to be declared before routes

restify runs handlers in the order they are registered on a server

restify ships with several handlers you can use

http://restify.com/#bundled-plugins

Routing


					server.RESTmethod('endpoint name', handler function);
 						

					server.get('/hello/:name', function(req, res, next){
						res.send('hello ' + req.params.name);
						next();
					});

Content Negotiation

restify automatically selects the content-type to respond with

restify ships with application/json, text/plain and application/octet-stream formatters


					res.setHeader('content-type', 'application/foo');
					res.send({hello: 'world'});
